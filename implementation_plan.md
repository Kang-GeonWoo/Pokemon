# 구현 계획: 포켓몬 한국 전략 허브 기능 통합

## 목표 설명
포켓몬 쇼다운(Showdown), 포켓DB 도쿄(Pokedb.tokyo), 로토포탈(RotoPortal), 게임위드(GameWith) 등 주요 커뮤니티 사이트의 핵심 기능들을 벤치마킹하여, 한국 사용자들의 입맛에 딱 맞는 'K-포켓몬 전략 허브'를 구축합니다. 단, 실제 배틀 시뮬레이터 기능보다는 정보 공유와 전략 수립에 집중합니다.

## 제안하는 새로운 기능 및 통합 계획

### 1. 지능형 팀 빌더 (쇼다운 / 포켓DB 스타일)
- **목표:** 전문가 수준의 GUI를 통해 자신만의 팀을 짜고 공유할 수 있게 합니다.
- **핵심 요소:**
    - 6마리 포켓몬 슬롯 관리.
    - 상세 편집: 도구, 특성, 성격, 노력치(슬라이더 사용), 개체값, 기술(4가지).
    - 스탯 자동 계산: 레벨(실전 기준 50), 기본 종족값, 노력치, 개체값, 성격 반영.
    - 쇼다운 텍스트 포맷 내보내기/가져오기 연동.
    - 한국어 이름 완벽 검색 지원 (이미 구축된 로컬라이징 DB 활용).

### 2. 메타 분석 및 랭킹 (포켓DB 도쿄 / 쇼다운 통계 스타일)
- **목표:** 현재 랭크 배틀 시리즈의 실시간 사용량 통계를 제공합니다.
- **핵심 요소:**
    - 포켓몬 사용률 랭킹 (Top 10 / Top 50 등).
    - 개별 포켓몬 상세 페이지:
        - 가장 많이 쓰이는 기술, 특성, 도구, 성격, 테라스탈 타입 통계.
        - 자주 함께 쓰이는 파트너 포켓몬 추천.
    - 데이터 소스: 쇼다운 API 데이터를 가공하여 시각화.

### 3. 렌탈팀 게시판 및 쇼케이스 (로토포탈 스타일)
- **목표:** 마스터볼 등급의 검증된 팀이나 입상 팀들을 공유하는 커뮤니티 공간.
- **핵심 요소:**
    - 유저 팀 등록 (팀 빌더와 연동).
    - 렌탈 코드(ID) 표시 기능.
    - 좋아요/추천 시스템 및 태그 필터링 (예: '시리즈 1', '더블배틀', '대면구축' 등).
    - 6마리 포켓몬 아이콘 시각화.

### 4. 전략 가이드 및 티어 리스트 (게임위드 스타일)
- **목표:** 초보자부터 고수까지 아우르는 정보 콘텐츠 제공.
- **핵심 요소:**
    - 포켓몬별 육성론 가이드 (예: '날개치는머리 베스트 샘플').
    - 마크다운 에디터를 활용한 고퀄리티 가이드 작성 기능.
    - 직관적인 '티어 리스트' 시각화 컴포넌트.

## 아키텍처 업데이트 계획

### [프론트엔드 UI/UX]
- **[신규] `apps/web/app/team-builder/page.tsx`**: 팀 빌더의 메인 인터페이스.
- **[신규] `apps/web/app/pokedex/page.tsx`**: 메타 분석 및 포켓몬 도감 목록.
- **[신규] `apps/web/app/teams/page.tsx`**: 렌탈팀 갤러리/게시판.
- **[수정] 공통 컴포넌트**: 노력치 슬라이더, 스탯 차트, 포켓몬 스프라이트 등을 위한 전용 컴포넌트 개발.

### [백엔드 / 데이터베이스]
- **[수정] DB 스키마**: 기존 `Team`, `TeamVersion`, `Post`, `PokemonUsage` 테이블을 적극 활용하여 API 제공.
- **[신규] API 라우트**:
    - `GET /api/teams` (렌탈팀 목록)
    - `POST /api/teams` (새 팀 등록)
    - `GET /api/stats` (메타 랭킹 정보)

## 단계별 구현 일정 및 상세 기술 계획

1.  **1.5단계: 팀 빌더 고도화 및 UI/UX 개선 (현재 목표)**
    - **Combobox 전면 도입:** 기존 `Select` 컴포넌트를 검색이 가능한 `Combobox` 컴포넌트로 전면 교체 (특성, 도구, 성격, 기술 모두 타이핑 검색 지원).
    - **지능형 자동화:** 포켓몬 선택 시 해당 포켓몬의 '첫 번째 특성'을 API에서 불러와 자동 할당.
    - **기술 호환성 경고:** 선택한 포켓몬이 배울 수 없는 기술인 경우, 빨간색 경고 아이콘 띄우기 (데이터 처리를 위해 PokeAPI 연동 검토).
    - **쇼다운 연동:** 한국어 기반 텍스트를 파싱하여 팀 슬롯에 채워넣는 `Import` 로직과 그 반대인 `Export` 로직 완성 (로딩 UI 필수 적용).
    - **AI 파티 솔루션:** 선택된 6마리의 타입 상성, 역할군을 분석하여 보완점과 추천 포켓몬을 제시해주는 피드백 패널 구현.
    - **인증 연동 준비:** 작성한 팀을 DB에 임시 저장하는 `Save` 기능 활성화.

2.  **2단계: 배틀 어시스턴트 고도화 (동적 예측)**
    - **6마리 선입력 기반:** 상대방 엔트리 6마리를 우선 입력받는 UI 구성. (6마리 조합에 따라 형태가 달라짐을 분석 로직에 반영).
    - **동적 재계산:** 예측된 기술 중 하나가 실제 배틀에서 확정(수동 선택)되었을 때, 해당 기술을 가지는 통계 집합으로 필터링하여 나머지 3개 기술의 확률을 실시간으로 다시 계산.

3.  **3단계: 커뮤니티 및 렌탈팀 시스템**
    - 로그인/회원가입 디자인 및 Supabase 연동.
    - 완성된 팀을 게시판에 렌탈 코드와 함께 자랑하는 갤러리 구현.

4.  **4단계: 메타 분석 및 전략 가이드**
    - 메인 대시보드에서 각 포켓몬의 사용률 랭킹 시각화.
    - 육성론 작성용 CMS 기능 및 티어 리스트 툴 완성.
