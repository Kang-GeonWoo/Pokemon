# Pokemon Showdown Assistant: Design System (Glassmorphism)

## 🎨 Theme: Neon Violet & Deep Indigo
- **Primary Color:** `#8B5CF6` (Violet-500)
- **Secondary Color:** `#10B981` (Emerald-500) for "Success" and "Predict" indicators.
- **Background:** `#0F172A` (Slate-900) - Deep, dark indigo.

## ✨ UI Style: Glassmorphism
- **Motto:** "Clear, Vibrant, and Intuitive."
- **Effect Tokens:**
    - `bg-white/10` with `backdrop-blur-md` for cards.
    - `border-white/20` for subtle boundaries.
    - Neon shadows for active/highlighted elements.

## 🌈 Components
- **Buttons:** Gradient backgrounds with subtle scale animation on hover.
- **Stats Table:** Zebra striping with opacity; high-usage Pokémon highlighted with a soft violet glow.
- **Battle Prediction:** Red/Green probability bars with progress animations.

## 📝 Typography
- **Headings:** `Outfit` (Modern, geometric)
- **Body:** `Inter` (Legible, technical)
- **Status:** All text in Korean, using polite and clear wording.
